This is a simple program for Solana enthusiasts.
It is simple Rust code for interacting with the Solana network.
The code is entirely open source.
Command-line arguments are used as input.

v1 - Displays your crypto wallet balance
Example:
cmd:> solana_wallet.exe *wallet_address* 

how to compile: 
cargo check 
cargo build --release

main code: solana_wallet\src\main.rs
All the comments in the code are in Russian, but I don't think translating them using AI will be a problem.
